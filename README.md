# MSFT Incidents in Practice Form

Static GitHub Pages app for the Mary Seacole Foundation Trust Incidents in Practice form.

## Deploy

Upload the contents of this folder to GitHub. For GitHub Pages use:

- Branch: `main`
- Folder: `/docs`

The app is duplicated as both `index.html` and `ip-form.html` so it can run standalone and also be linked from the incident-report app using `./ip-form.html`.

Build marker: `MSFT_IP_FORM_STANDALONE_CHECKED_20260608`.
